Como é uma estrutura básica de um HTML?

HTML (HyperText Markup Language) é a linguagem usada para criar páginas web. Sua estrutura básica segue uma hierarquia de elementos que organizam o conteúdo e definem como ele será exibido no navegador.

Esqueleto Básico do HTML

< !DOCTYPE html >
< html >
< head >
< meta charset="UTF-8" >
< title >Título da Página< /title >
< /head >
< body >
< h1 >Bem-vindo ao HTML!< /h1 >
< p >Este é um exemplo de estrutura básica.< /p >
< /body >
< /html >

Descrição dos Elementos

  1. < DOCTYPE html >
    • - Indica ao navegador que o documento está usando HTML5 (a versão mais moderna do HTML).
  2. < html >
    • Elemento raiz que envolve todo o conteúdo da página.
    • Contém dois blocos principais: head e body.
  3. < head >
    • Inclui informações meta sobre o documento, como codificação de caracteres, título e links para arquivos externos (ex.: CSS e JavaScript).
    • Não exibe conteúdo diretamente ao usuário.
    • Exemplos:
      • title: Define o título exibido na aba do navegador.
      • meta charset="UTF-8": Define a codificação de caracteres para suportar textos em várias línguas.
  4. < body >
    • Contém o conteúdo visível da página, como textos, imagens, links e outros elementos interativos.
    • Exemplos de elementos comuns:
      • h1 a h6: Cabeçalhos, do maior (h1) ao menor (h6).
      • p: Parágrafos.
      • img: Imagens.
      • a: Links.

Outros Elementos Importantes

  • Estruturas de lista:
    • < ul >: Lista não ordenada (com marcadores).
    • < ol >: Lista ordenada (numerada).
    • < li >: Itens das listas.
  • Tabelas:
    • < table >: Cria uma tabela.
    • < tr >, < th >, < td >: Linhas e Colunas da tabela.

Resumo

Um documento HTML começa com a declaração < !DOCTYPE html > e é dividido em duas partes principais: - < head >: Contém informações auxiliares e configurações. - < body >: Contém todo o conteúdo exibido ao usuário.

Essa estrutura serve como base para criar páginas web, que podem ser enriquecidas com CSS (para estilo) e JavaScript (para interatividade).
